## 🎯 Project Overview

InternML is a Machine Learning based system designed to evaluate student internship performance and predict their success probability.

The system takes student input data and provides:

- Performance Score  
- Success Prediction  
- AI-based Feedback  
- Certificate Eligibility  

---

## 🧠 Features

- 📊 ML-based performance prediction  
- 🤖 AI-generated feedback  
- 📄 PDF report generation  
- 🌐 Interactive Streamlit dashboard  
- 🗂️ Dataset upload & analysis  
- ☁️ MongoDB integration
